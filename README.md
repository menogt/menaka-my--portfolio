# Meno Pixels Final Red/Black Portfolio

This ZIP includes your real uploaded design images, video thumbnails, YouTube links, and logos.

## What was added

- MP logo
- Meno Pixels long logo
- 9 design/photo portfolio items
- 9 video portfolio items
- YouTube links embedded/openable in modal
- 1:1, 3:4, and landscape design showcase
- Red/black Claude-style premium header
- Focused positioning: Video Editor & Visual Designer

## Edit portfolio

Open `js/data.js`.

## Replace or add files

- Design images: `assets/images/designs/`
- Video thumbnails: `assets/images/video-thumbnails/`
- Logos: `assets/logos/`

## Deploy on Netlify

Extract this folder and drag it into Netlify.
